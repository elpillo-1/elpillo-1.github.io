Siglas

LAN

Local Area Network: una red de computadoras que cubre un área geográfica pequeña, como una oficina o un edificio.

IP

Internet Protocol: protocolo de comunicación utilizado para la identificación y localización de dispositivos en una red.

TCP

Transmission Control Protocol: protocolo de comunicación que garantiza la entrega confiable de datos en una red.

UDP

User Datagram Protocol: protocolo de comunicación que permite la transmisión rápida de datos sin garantizar su entrega.

DNS

Domain Name System: sistema que traduce nombres de dominio en direcciones IP.

HTTP

Hypertext Transfer Protocol: protocolo utilizado para la transferencia de información en la web.

HTTPS

Hypertext Transfer Protocol Secure: versión segura de HTTP que usa cifrado SSL/TLS.

MAC

Media Access Control: dirección única asignada a una tarjeta de red para identificarla en una red.

VPN

Virtual Private Network: tecnología que permite una conexión segura y cifrada a través de una red pública.

DHCP

Dynamic Host Configuration Protocol: protocolo que asigna direcciones IP automáticamente a dispositivos en una red.

FTP

File Transfer Protocol: protocolo utilizado para transferir archivos entre dispositivos en una red.
